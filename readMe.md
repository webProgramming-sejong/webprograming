# Sejong Web Programming

## To Start

### 1. create node_module

#### Open terminal in Vscode and run below instruction

    npm install

### 2. Run Dev server 

#### Open terminal in Vscode and run below instruction ( Available instruction sets are declared in package.json)

    npm run dev
    
### 3. Enjoy
